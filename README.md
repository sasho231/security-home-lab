# Security Home Lab

Hands-on cybersecurity architecture lab built on Windows 10 / WSL2.
Implements ZTA, KSPM, runtime threat detection, SIEM, and DevSecOps
using open-source tooling — zero cloud cost.

## Stack
- **Kubernetes**: kind (local cluster)
- **Runtime Security**: Falco + Falcosidekick
- **KSPM**: kube-bench, Trivy, kube-hunter
- **SIEM**: Wazuh
- **Identity / ZTA**: Keycloak, HashiCorp Vault
- **DevSecOps**: GitLab CE, Semgrep, OWASP ZAP
- **IaC**: Terraform (Azure), Kubernetes YAML manifests

## Certifications
SC-100 | SC-200 | SC-300 | AZ-500

## Lab Sessions
Each folder contains working configs with explanations.
